# 회고
코딩 테스트를 보면서 기본적인 실수를 많이 해서 많이 속상했다. 매번 생각해보면 for 루프에서 idx를 잘못 파악해서 마지막 1번이 고려되지 않거나 애초에 이상한 loop를 도는 식으로 작성하는 것 같다.
이를 위해서 코딩 테스트를 기본적인 것부터 실수 없이 푸는 연습을 해야할 것 같다.

사이드 프로젝트를 진행하면서 로컬에서 Ollama를 이용하여 Agent AI 기반 MCP를 구축했다. 이 과정에서 아래와 같은 더 고차원적으로 해결해야할 부분을 발견했다.

1. 데이터 환각 문제를 어떻게 해결할 것인가?
2. 분산 환경에서 중복 AI Call이나 중복 데이터를 어떻게 처리할 것인가?

1번 문제는 LLM을 더욱파야하기 때문에 관련 학습이 필요할 것 같다. 하지만 2번 문제의 경우 낙관적 락이나 분산 락을 이용하여 쉽게 해결할 수 있을 것 같다.
사이드 프로젝트에선 기능 추가보다 이 부분을 더욱 깊게 파보고자 한다.
